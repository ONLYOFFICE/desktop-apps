ONLYOFFICE 5.5.1 - April 1, 2020

New features

All editors

  • Loading speed improvements.
  • Symbol table now is a system component, not a plugin.
  • Added a new button: Top Toolbar -> Collaboration -> Remove comments.
  • The Default Size button in the image settings was replaced with the Actual Size one.

Document Editor

  • Added the possibility to remove table cells.
  • Added the possibility to insert several rows/columns into the table.
  • Added the possibility to add captions for images, shapes, tables and equations.
  • Added new content control types (Picture, Combo box, Drop-down list, Date, Checkbox).
  • Added the possibility to compare documents (available only for the paid version).
  • Added new options for bulleted and numbered lists.
  • Added the possibility to draw and erase a table.
  • Added the possibility to edit gutter and mirror margins.

Spreadsheet Editor

  • Added the possibility to recalculate all formulas.
  • Added new scale options.
  • Added new options for cell fill.
  • Added the possibility to set Cell Snapping for images, shapes and charts.
  • Added the possibility to work with several sheets at once.
  • Added the Hungarian localization to the regional settings.
  • Added the bulleted and numbered list options to the context menu.
  • Added the possibility to change the bullets marker.
  • Added new spellchecker options.
  • Added the possibility to sort by several columns/rows.
  • Added a new option for setting separators.

Presentation Editor

  • Added the possibility to add objects (tables, shapes, charts, images) to a slide template.
  • Added the possibility to reset a slide.
  • Added new list settings.
  • Added the possibility to add objects (images, charts, tables) to a placeholder.

Fixes

Document Editor

  • Fixed an issue with comment duplication in other editors (bug #44689)

Spreadsheet Editor

  • Better support of TIME function (bug #44849)

All editors

  • Fixed some security issues
  • Fixed an issues with DOC, RTF and XLSX formats(bug 44756) (bug 44934) (bug 44840)
  • Fixed corrupting of mime-type after some DOCX editing (Bug 44957)

and lots other fixes in all editors
