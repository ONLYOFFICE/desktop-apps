ONLYOFFICE 5.3.3 - July 9, 2019

New features

All editors

  • Completely rewritten the font engine, adding a better support of Chinese, Japanese and Korean fonts and much more;
  • Added the possibility to flip and rotate shapes and images;
  • Added the possibility to crop images;
  • Adding a bookmark does not close the bookmark window;
  • The comments are now shown in all editing and viewing modes;
  • Added the new hotkeys on macOS and their description in documentation;
  • Added options for objects alignment (align to page, margin, slide);
  • Added new hints for shapes (bug #20091);
  • Completely rewritten the photo editor plugin;
  • Updated the interface translations to all the main languages available in the editors;
  • Updated the documentation articles;

Document Editor

  • Added the support for calculating tabs in Word 2013 style positioned to the right of the right margin;
  • Added formulas in tables;
  • Added the possibility to save as DOTX, PDF-A, OTT;
  • The review changes are now shown in view mode;
  • Added the possibility to prevent deletion of the review changes of another user;
  • Added the possibility to prevent deletion of the comments of another user;
  • Added the description for moved text and move to changed text in review mode;
  • Added the possibility to get link to bookmark;
  • Added the possibility to automatically copy the selected text to the search window;

Spreadsheet Editor

  • Added the bg-BG, sv-SE and sv-FI number formats;
  • Translates the functions and their descriptions to Italian;
  • Replaced the behavior of the mouse click for the links in cells;
  • Added the ASC, BETAINV, HYPERLINK functions;
  • Added the support of print areas;
  • Added the possibility to set the exact text orientations in degrees;
  • Added the text to column wizard;
  • Added the paste text wizard;
  • Added the possibility to save as XLTX, PDF-A, OTS;
  • Added the support of the array formulas;
  • Added the support for the gradient and texture cell fill;
  • A hint is now shown with function and argument descriptions while typing;
  • Added the support to show minimum and maximum values in the status bar;
  • Added the possibility to distribute objects;
  • Rewritten the cut cell process (cell is not removed after cutting);

Presentation Editor

  • Added the possibility to save as POTX, PDF-A, OTP;
  • Added the font rendering option;
  • Added an option to search and replace the text;
  • Added the possibility to enter the slide number manually for the internal hyperlinks;

Bugfix

All editors

  • Fixed a lot of bugs with drawings positioning;
  • Fixed more problems with SVG;
  • A lot of bugfixes in all product modules;

Document Editor

  • Fixed the problem with calculating the numbering value within merged cells (bug #41699);
  • Fixed the issue with deleting the whole TOC field after selecting it (bug #41714);
  • Fixed the issue with showing the numbering as changes for review in specific documents (bug #41518);
  • Fixed the crash which occurred when refreshing the recalculated data;
  • Fixed the bug with moving object in group;
  • Fixed the problem with updating positions in the inner classes within a paragraph (bug #41848);
  • Fixed the issue when records were added to reviews with changes of the text properties if in fact there have been no changes (bug #41833);
  • Fixed the issue with showing the changes from the second user if the display of the changes was disabled (bug #41576);
  • Fixed the problem with accepting/rejecting review changes;

Spreadsheet Editor

  • Fixed the issue with undoing the moved cell in print area (bug #41723);
  • Fixed the JavaScript error which occurred for Replace all in some cases (bug #41760);
  • Fixed the column index error which occurred during insertion;
  • Fixed the issue with adding the changes to the history on drawing sparklines;

and lots other fixes in all editors
