ONLYOFFICE 5.2 - 21 декабря, 2018

Новые функции

  • Добавлена настройка для изменения имени пользователя;
  • Улучшена работа с HiDPI мониторами;
  • Расширен список провайдеров порталов для совместной работы;
  • Добавлена опция шифрования документов (в тестовом режиме);

Во всех редакторах

  • Обновлены переводы интерфейса на все основные языки, доступные в редакторах;
  • Добавлена возможность настройки начального масштаба для встроенных редакторов;
  • Добавлена возможность настройки доступности справки в редакторе;
  • Добавлено отображение вкладок Файл и Плагины для просмотрщиков;
  • Имя пользователя теперь выделяется цветом в комментариях, изменениях при рецензировании, сообщениях чата;
  • Устранено дублирование пользователей, находящихся онлайн, в левой панели чата;
  • Добавлена возможность сортировки комментариев во всплывающем окне по времени создания в порядке возрастания;

Редактор документов

  • Добавлены настройки для элементов управления содержимым (выделение цветом и настройки внешнего вида);
  • Изменения при рецензировании и комментарии теперь отображаются в объединенном окне;
  • Добавлены пресеты для размеров страницы А0, А1, А2, А6 (bug #36583);
  • Изменен формат кодировки для файлов TXT (bug #36998);
  • Добавлен режим для заполнения форм;
  • Разрешено закрытие диалогового окна диаграммы при загрузке (bug #36870);
  • Разрешено закрытие окна при сохранении в формат TXT;
  • Разрешена вставка фигур, когда фигура выделена;
  • Новые ревизии теперь можно проверить в Быстром режиме совместного редактирования;
