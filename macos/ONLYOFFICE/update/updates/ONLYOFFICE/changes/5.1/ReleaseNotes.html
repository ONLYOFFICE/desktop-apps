ONLYOFFICE 5.1 - May 18, 2018

New features

All editors

  • Redesigned app interface;
  • Ability to replace image via context menu;
  • Fully rewritten composite input for characters;
  • New no squares font engine, find best replacement font for □ characters;
  • Ability to distribute data in tables;
  • Support of shape side panel for images;
  • Change table size by drag'n'drop;
  • New bullet list marker –;
  • Added Macros plugin;
  • Search in help;
  • Speedup of opening DOCX, XLSX, PPTX files;
  • Better compatibility with OpenFormat, RTF;

Document Editor

  • Implement an East Asian script and line break in hieroglyphs;
  • Ability to set Tab Leader symbols;
  • Support of Table of Contents;
  • New Navigation left sidebar;
  • New Reference tab;

Spreadsheet Editor

  • Add new formulas: CONVERT, FTEST, HYPGEOM.DIST, F.TEST, FORECAST.ETS, FORECAST.ETS.CONFINT, FORECAST.ETS.SEASONALITY, FORECAST.ETS.STAT, FORMULATEXT, IFS, PDURATION;
  • New None Table Template;
  • New editing tools for Pivot Tables;
  • New regional presets - Deutsch (Schweiz), Español (México), Nederlands (Nederland), Slovenčina (Slovenská republika);
  • New date formats yy/m/d, yy/mm/dd, yyyy/m/d;
  • CSV preview before opening;
  • Add French translation for formulas;
  • Add Spanish formula translations;
  • Ability to select Cell format via context menu;
  • Support a lot of features in XLS format (macros, controls etc.)

Presentation Editor

  • Add presenter view mode;
  • Special paste;
