ONLYOFFICE 5.1 - 18 мая, 2018

Новые функции

Во всех редакторах

  • Переработан интерфейс приложения;
  • Возможность замены изображения через контекстное меню;
  • Полностью переработан композитный ввод символов;
  • Добавлен новый механизм отображения шрифтов без квадратов, который подбирает наиболее подходящий замещающий шрифт для символов □;
  • Добавлена возможность распределения данных в таблицах;
  • Добавлена поддержка боковой панели параметров фигуры для изображений;
  • Добавлена поддержка изменения размера таблиц с помощью перетаскивания;
  • Добавлен новый маркер для маркированных списков: –;
  • Добавлен плагин Macros;
  • Добавлен поиск по справке;
  • Ускорение открытия файлов в формате DOCX, XLSX, PPTX ;
  • Улучшена совместимость с форматами OpenFormat, RTF;

Редактор документов

  • Реализована поддержка восточноазиатского письма и разрыв строки в поддержке иероглифов;
  • Добавлена возможность задавать символы заполнителя для позиций табуляции;
  • Добавлена поддержка Оглавления;
  • Добавлен новый элемент Навигация на левой боковой панели;
  • Добавлена новая вкладка Ссылки;

Редактор электронных таблиц

  • Добавлены новые функции: CONVERT, FTEST, HYPGEOM.DIST, F.TEST, FORECAST.ETS, FORECAST.ETS.CONFINT, FORECAST.ETS.SEASONALITY, FORECAST.ETS.STAT, FORMULATEXT, IFS, PDURATION;
  • Добавлен новый шаблон таблиц None;
  • Добавлены новые инструменты редактирования для сводных таблиц;
  • Добавлены новые региональные настройки: Deutsch (Schweiz), Español (México), Nederlands (Nederland), Slovenčina (Slovenská republika);
  • Добавлены новые форматы даты: yy/m/d, yy/mm/dd, yyyy/m/d;
  • Добавлен предварительный просмотр файлов CSV перед их открытием.
  • Добавлены переводы функций на французский и испанский языки;
  • Добавлена возможность выбирать формат ячейки через контекстное меню;
  • Добавлена поддержка множества функций для формата XLS (макросы, элементы управления и так далее);

Редактор презентаций

  • Добавлен режим докладчика;
  • Добавлена специальная вставка;
